#  Stock Market Prediction using Time Series Analysis

## Overview
This project analyzes historical stock prices of Tata Consultancy Services (TCS) and predicts future stock trends using Time Series Analysis and Machine Learning techniques.

Models implemented:
* ARIMA
* Random Forest Regression

Evaluation metrics:
* Mean Absolute Error (MAE)
* Root Mean Squared Error (RM
